CÓDIGOS EN EL GITSHELL

Al inicio (Directory)
.git

Si el directorio ya es Documents Github solo escribo
ls

Escojo carpeta repositorio
cd NombreCarpetaRepositorio

Para comprobar error
git status

Para sincronizar
git pull

Para agregar
git add *

Para comentar
git commit -m "COMENTARIO AQUI"

Para subir lo agregado y comentado
git push
